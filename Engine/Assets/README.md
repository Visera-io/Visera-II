# Visera Assets

## Icons
- 16x16 (The classic: for file explorer details view, browser address bars)
- 24x24 (Occasionally used in Explorer lists)
- 32x32 (Standard for file explorer, taskbar in older Windows versions)
- 48x48 (Windows Explorer medium icon view)
- 64x64 (Windows Explorer large icon view, rarely used but good to have)
- 128x128 (Windows Explorer extra large icon view)
- 256x256 (The modern standard for application icons, used in Start Menu, Desktop shortcuts, and macOS)
